## Getting Started

### Install Eleventy

```bash
npm install -g @11ty/eleventy
```

### Run Eleventy

```bash
eleventy --serve
```

Now navigate to [http://localhost:8080](http://localhost:8080) to see your site.

Hot reloading is enabled, so any changes you make to your files will be reflected in the browser.
